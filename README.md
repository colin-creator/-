# 生信刷题神器

这是北京协和医学院应用生物信息学课程中出现过的习题整理，不懂考试会不会有原题，但是我就复习这些了，希望老师高抬贵手。

## 在线访问

本项目已启用 GitHub Pages，可以直接在线访问刷题：

🔗 **访问地址**: https://colin-creator.github.io/-/

> 注：仓库名称就是 `-`（一个横杠），所以 URL 就是这样的。

## 本地使用

也可以直接下载 `index.html` 文件到本地，用浏览器打开即可使用。

## 功能特点

- 📱 移动端友好的界面设计
- 🔄 支持顺序刷题和随机刷题两种模式
- ✅ 即时答案反馈
- 📊 学习进度跟踪
